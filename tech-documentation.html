Start

React is a JavaScript library for user interface development.

On this page, we'll take a quick look at the documentation and other resources that may come in handy when using React for the first time.

React was designed from the ground up to be rolled out gradually. In other words, you can start small and only use the React functionality you need at the moment. The information in this section will be useful in any situation: when you first get to know React, when creating a simple dynamic HTML page, or even when designing a complex React application.

Adding React to the Site

You can add React to your site in literally one minute. After that, you can place several dynamic widgets on the site and gradually increase the use of React in your project.

Step 1: Adding a DOM-container to HTML

First, open the HTML file of the page you want to edit. Add an empty tag <div> in the place where you want to render something with React. For instance:

<!-- ... HTML code ... -->
<div id="like_button_container"></div>
<!-- ... HTML code ... -->

Then assign the created <div> unique id attribute. This will allow you to later find the tag from JavaScript code and display the React components inside it.

Step 2: Adding script tags

Now add three <script>-tag before the closing tag <body>:

<!-- ... HTML code... -->
<!-- Load React. -->
<!-- Note: when deploying to production, replace "development.js" with "production.min.js". -->
<script src="https://unpkg.com/react@17/umd/react.development.js" crossorigin></script>
<script src="https://unpkg.com/react-dom@17/umd/react-dom.development.js" crossorigin></script>
<!-- Load our React-component. -->
<script src="like_button.js"></script>
</body>

The first two tags load React. The third tag loads the code for your own component.

Step 3: Create React-component

Create a file called like_button.js next to your HTML file. Take this starter code and paste it into the file you created earlier.

'use strict';
const e = React.createElement;
class LikeButton extends React.Component {
constructor(props) {
super(props);
this.state = { liked: false };
}
render() {
if (this.state.liked) {
return 'You liked this.';
}
return e(
'button',
{ onClick: () => this.setState({ liked: true }) },
'Like'
);
}
}

Add 2 more lines to the end of the like_button.js file, after the start code:

// ... code before ...
const domContainer = document.querySelector('#like_button_container');
ReactDOM.render(e(LikeButton), domContainer);

These two lines of code are looking for an element <div>, which we added in the first step, and then render a React component with a Like button inside it.

Ready!

That's it! You have just added your first React component to the page.
